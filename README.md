### Vue 项目结构

```
项目（不定期更新）
|--public					css、js、附件等静态文件目录
|--src
	|--api   				接口定义
	|--assets				资源文件
	|--common				普通js代码，如
	|--components			各种vue组件
	|--const				常量
	|--directives			指令
	|--filters   			过滤器
	|--router    			路由定义
	|--store 				数据中心
	|--style 				项目样式
	|--ui					项目自定义UI库
	|--utils 				工具类
	|--views 				视图组件
	|  App.vue
	|  main.js
|  .browserslistrc
|  .editorconfig
|  .eslintrc.js  				eslint代码规则配置
|  .gitignore
|  babel.config.js
|  jsconfig.json
|  package-lock.json
|  package.json
|  README.md
|  vue.config.js 				vue配置文件
```
